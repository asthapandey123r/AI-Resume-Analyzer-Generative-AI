# AI-Based Resume Analyzer using Generative AI

## 📌 Project Overview
This project uses NLP and Generative AI to analyze resumes against job descriptions.
It calculates a semantic match score and provides AI-generated suggestions to improve resumes.

## ⚙️ Features
- PDF resume text extraction
- Resume–job description semantic comparison
- Match score generation using cosine similarity
- AI-based resume improvement suggestions

## 🛠 Tech Stack
- Python
- NLP
- Sentence Transformers
- Generative AI
- PyPDF2

## 🚀 How It Works
1. Upload resume (PDF)
2. Extract text from resume
3. Generate embeddings using Sentence Transformers
4. Compute similarity score
5. Generate AI-based suggestions
